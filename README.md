# ArchiLink

ArchiLink is a platform that connects Filipino architects and clients through verified profiles, portfolio showcasing, and project-based hiring.

## Features
- PRC license verification  
- Architect portfolios  
- Project posting and applications  
- In-app messaging  

## Tech Stack
- TypeScript  
- React (TSX)  
- JSON
